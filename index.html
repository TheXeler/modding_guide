Stonehearth 模组制作指南

欢迎来到 Stonehearth 模组制作指南！

本模组制作指南采用循序渐进的"操作方法"方式编写。我们将其分为多个组别，从基础到高级。其中还包括对游戏中各类脚本的说明。

在本指南中，我们将详细讲解模组制作的各个环节，如何从零开始创建模组，以及如何利用工具提高模组制作效率。

什么是模组制作

根据维基百科的定义，模组制作是"对电子游戏程序代码进行修改，使其以不同于原始版本的方式运行"。Stonehearth 在设计之初就支持模组制作。

入门建议

在深入探讨模组制作的具体细节之前，仔细规划你想要添加到模组中的内容总是明智之举。 有时想法太多，难以确定从何入手。经过数小时、数天或数周后，现实生活的琐事可能会让你失去兴趣或没有时间继续。通过将所需功能进行规划并分解成小块，你将更有可能成功完成模组。从小处着手，逐步扩展，永远是更好的选择。

一切始于设计和规划。花点时间将你的想法写下来，明确你希望游戏如何表现。在了解了模组 API 的各个系统后，思考实现目标的不同方法，并将其分解为小任务。

如果你感到不知所措，可以寻求其他模组制作者的帮助。例如，有些用户擅长制作精美的模型或动画，但在编程方面遇到困难，反之亦然。许多优秀的模组都是由两人或更多人组成的团队完成的。欢迎在我们的论坛分享你的想法或提出问题：https://discourse.stonehearth.net/c/modding
