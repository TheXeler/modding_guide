模组制作工具

制作模组需要什么工具？

  • 任意文本编辑器。你可以使用任何喜欢的编辑器，从记事本到 Sublime Text 都可以。如果文本编辑器具有帮助编程的功能（如语法高亮等）会更好，但你也可以简单地使用 Windows 自带的默认编辑器（记事本）。

实际上，这是唯一必需的工具，但根据你的模组功能，你可能需要更多工具：

  1. 如果你想向游戏中添加自己的模型，你需要一个3D像素编辑器。有免费的编辑器，如 MagicaVoxel、VoxelShop 或 Goxel；也有付费的编辑器，如 Qubicle。它必须能够导出 .qb 格式的文件。此外，如果你想为游戏中的模型制作动画，它还需要能够导出为其他 3D 格式（.obj 等），并且能够使用"矩阵"或模型中的不同图层，以便分离不同的部分用于动画制作。

  2. 对于动画制作，你可以使用免费程序如 Blender，或付费程序如 3ds Max 或 Maya（这些程序也有免费版本）。你需要某种插件或扩展，能够将骨骼和动画导出为游戏使用的 JSON 结构。

  3. 对于检查 JSON 语法，有一个方便的在线工具叫 JsonLint，还有另一个用于比较两个 JSON 文件内容的工具：Json Diff。

  4. 为了更快地创建模组，可以使用官方的 SHED（StoneHearth 编辑器）。它现在包含用户友好的粒子效果编辑器、战役自动完成功能和配方编辑器。它捆绑在游戏文件夹中，位于 /stonehearth/shed/。

  5. 为了更轻松地翻译现有模组到其他语言，有一个网页选项 Stonehearth-Translator，或者可下载的应用程序 thk 的工具。
