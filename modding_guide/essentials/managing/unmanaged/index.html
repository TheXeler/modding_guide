本地模组

我们将必须由玩家手动安装和更新的模组称为"本地"模组。 Steam 创意工坊模组会自动更新，并且上传需要额外步骤，因此它们在本指南中有自己的页面。

  • 卸载

  • 上传建筑模板

手动安装模组的说明：

当你手动下载新模组时，它通常是 ".smod" 文件格式。这是一个包含模组运行所需所有内容的压缩文件。下载此文件并将其放入 Stonehearth 安装目录的 "mods" 文件夹中：

  • 如果使用 Humble Bundle 安装程序安装游戏，通常位于 C:\Program Files (x86)\Stonehearth\mods。
  • 如果通过 Steam 安装，则 mods 文件夹位于 C:\Program Files (x86)\Steam\steamapps\common\Stonehearth\mods。

将模组放入此文件夹后，你只需启动游戏，模组就应该处于激活状态。 如果没有，请从主菜单进入模组菜单，并确保在"本地模组"列表中勾选了该模组。

其他格式

如果模组提供的扩展名不是 ".smod"（如 ".zip" 或 ".rar"），你需要解压它。

游戏可以读取 ".smod" 格式或普通文件夹格式的模组。如果你同时拥有同一模组的这两种格式，它将优先读取普通文件夹。

如果之后你更新了 .smod 文件，但仍保留普通文件夹，你将遇到冲突，因为游戏将加载不包含最新版本模组的普通文件夹。这对于 stonehearth 模组尤其重要，如果你同时拥有普通文件夹和 .smod 格式，并且 Steam 更新了游戏，你可能会遇到错误，并且看不到新内容。删除普通文件夹以解决此问题。

确保如果解压模组，它不会创建同名的内部文件夹，否则游戏将无法正确加载模组。将文件复制并粘贴到外部目录以解决此问题。

手动卸载模组的说明：

如果你从外部网站手动下载了模组，它将位于你的 mods 文件夹中。 关闭游戏后，导航到该目录并删除相应的模组文件夹或 .smod 文件。然后重新启动游戏。

请记住，如果你尝试在卸载后加载使用该模组创建的存档，将会出现错误（禁用模组而不是删除它也会出现同样的情况）。

手动上传模组的说明：

完成模组制作后，是时候打包它以方便分享了。在 mods 文件夹中，选择你的模组文件夹，右键单击并压缩它。

然后重命名它，将扩展名从 .zip 改为 .smod（更改扩展名时 Windows 会发出警告，只需点击"是"）。这样玩家可以更快地安装。

再次仅使用 smod 文件进行测试，以检查它是否能正常加载。

压缩模组文件夹后不要重命名 zip/smod 文件。清单文件可能会因为内部路径不匹配而变得无效。

然后，将其上传到你选择的地方。

如果你计划将模组上传到 Steam 创意工坊，你不应该打包它，上传按钮会处理这个问题（此外你需要先为你的模组生成 Steam 文件 ID）。请参阅指南的此部分以获取更多信息。

要更新本地模组，你需要手动用新版本替换整个 .smod 文件/普通文件夹。

将建筑模板作为模组上传的说明：

模组菜单提供了一种简单的方法来将建筑模板包作为模组重新分发。
如果你以 .smod 格式下载建筑模板模组，你需要解压它们以便游戏能够读取模板。当前版本不支持从压缩模组中读取模板。确保解压后它们具有正确的结构。

首先创建一个带有基本清单文件的模组。然后，在你的模组文件夹中创建一个名为 "building_templates" 的文件夹：

在你的清单文件中，在 "info" 部分添加 "is_building_template_mod" : true 并保存更改（记得在需要的地方添加逗号）：

  {
     "info": {
        "name": "My Templates Mod",
        "namespace": "my_templates",
        "version": 3,
        "is_building_template_mod" : true
     }
  }

现在在模组菜单中，将鼠标悬停在你的模组上并点击"编辑此模组的模板列表"按钮，该按钮带有编辑模板的图标。 将出现一个新窗口，让你从现有的建筑模板中选择： 模板将被复制到你的模组中，但不会从主游戏中删除。如果你以后决定添加其他模板，或从模组中删除某些模板，你也可以通过此菜单进行操作（只有你模组中的模板会受到删除的影响）。

点击"确定"后，将出现确认对话框： 在模组列表中，具有建筑模板的模组右侧将有一个箭头按钮。点击它将展开/折叠模组拥有的模板列表（如果你需要查看哪些模板在哪个模组中，这很方便）：

最后，记得按照上述部分的说明压缩你的模组并上传。下载你的模板模组的玩家需要先解压它。
如果你压缩了你的模组，你将无法从模组菜单中查看/编辑其模板。首先你需要解压它。

游戏将从所有未压缩的模组中收集模板，并在建筑编辑器的模板列表中显示它们。
