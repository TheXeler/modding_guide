编程语言

Stonehearth 的核心引擎使用 C++11 和 OpenGL 编写。我们使用 Horde3d 引擎进行渲染，并添加了一些自定义节点类型、自定义扩展以及大量我们自己的着色器。我们使用 Lua 编写所有游戏玩法和 AI。最后，我们使用 JavaScript 和 HTML 构建用户界面。

我们使用 Qubicle 制作所有体素艺术。Stonehearth 原生支持 .qb 文件，我们的资源文件格式只是 JSON，因此将新的艺术资源添加到游戏中非常容易。我们使用 3DSMax 进行骨骼绑定和动画制作（后来我们切换到 Maya），但动画文件格式再次只是 JSON。2D 艺术是 .png 文件，使用 Photoshop 创建。

仅使用 JSON 就可以轻松为 Stonehearth 创建模组，并且有各种免费工具可用于创建 2D/3D 艺术。
