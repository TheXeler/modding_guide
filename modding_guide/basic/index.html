基础 Modding

在本指南中，我们将提供并解释基础 modding 示例，面向那些想要了解如何更改或向游戏添加特定内容的玩家。 中级和高级指南涵盖更复杂的 modding。如果您是 Stonehearth modding 的新手，建议在深入探讨更高级的主题之前，先从做一些简单的事情开始，以了解游戏的结构和 modding API 提供的不同工具。

您可以在此处查看与艺术相关的指南这里。

在尝试任何 modding 之前，请阅读基础指南。
