添加新物品

我们已经看到 Stonehearth 中的所有事物都由实体和组件构成。在这里，我们将介绍如何创建新物品。

正如我们之前所说，大多数常规模组制作是通过将文件从 Stonehearth mod 复制粘贴到您自己的 mod 中，然后修改它们来完成的。

为了制作可放置的物品，我们需要了解 entity forms 组件。
您可以手动或使用 SHED 创建物品。
物品的比例可以在其 JSON 文件中更改。

  • 实体形式
  • 手动添加物品
  • 使用 SHED 添加物品
  • 更改物品比例

最后说明

创建新物品后，我们可以创建一个配方，以便可以制作它，或将其用作怪物掉落物，或使其仅在商店中可用等。 当然，如果我们愿意，还可以为其添加更多组件。

其他最常用的组件将在基础/中级指南的后续部分中介绍。
