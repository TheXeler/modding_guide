针对自定义实体

在这里，我们将讨论如何使用 Blender 插件为自定义实体创建动画（当我们无法重用 stonehearth mod 中的现有 rig 时）。

  • 创建骨骼
  • 创建 meta 文件
  • 为自定义实体创建动画

首先，确保您已经安装了 Blender 和插件，并准备了场景（始终确保从干净的场景开始）。按照这里的说明将您的自定义模型导出为 .obj。

创建骨骼

为了为我们的自定义实体制作动画，我们需要从头创建骨骼和 meta 文件。如果您只有少量骨骼并且可以猜测它们的枢轴应该在哪里，可以手动完成，但在这里我们将解释如何在 Blender 插件的帮助下完成：

  1. 在 Blender 中导入您的自定义模型的 .obj 文件，并确保使用正交视图，以便更容易定位枢轴。

  2. 检查右上角层次结构中的骨骼名称是否正确，以防它们未正确导出。

  3. 所有身体部位的原点/枢轴都在错误的位置，因此现在我们必须将它们移动到正确的位置。

    在我们的示例中，我们有一个基于奶油圈饼的怪物。当选择其不同的身体部位时，所有原点都在 (0, 0, 0)，因为那是 Qubicle 1 中原点的位置。

    • 为此，我们需要显示多个视图。要做到这一点，请从右上角点击并拖动小三角形到视图，要么向左要么向下。

    • 然后将鼠标悬停在每个视图上，并按 Numpad 1、Numpad 3 和 Numpad 7 以显示模型的前视图、侧视图和顶视图。或者，如果您的键盘没有小键盘，请使用每个窗口的 View 菜单。

    • 有几种方法可以将光标移动到我们希望枢轴所在的位置：

      通过左键单击移动它，但它不会精确停留在我们希望的位置（查看不同的视图）。 如果您按 N 打开隐藏菜单，将有一个 3D 游标部分，您可以在其中手动设置 3D 游标的位置。这将确保它正好在我们想要的位置。 或者，进入编辑模式，右键单击网格的顶点、边或面，按 Shift+S 并点击'Cursor to selected'；然后返回对象模式：

    • 现在转到左侧菜单的 Tools 选项卡，然后点击'Set Origin'。点击'Origin to 3D cursor'选项。或者，按 Shift+Ctrl+Alt+C，选项将出现在鼠标旁边：

    • 此身体部位的橙色小点应从 (0, 0, 0) 移动到 3D 游标所在的位置。尝试旋转该身体部位，它应使用其新原点作为枢轴：

    • 对所有身体部位重复上述步骤。您可能会发现某些部分不像您希望的那样移动/旋转，在这种情况下，返回 Qubicle 并调整任何矩阵/层，重新导出为 .obj 并再次执行所有步骤，直到您满意为止。正确设置枢轴对于能够为您想象的模型制作动画非常重要。

  4. 添加您可能需要的任何辅助骨骼。至少，添加一个名为 "root" 的骨骼，位于实体的底部。

    要做到这一点，请选中 Scene 菜单的 Stonehearth UI 部分中的'Custom helper bones'复选框。

    输入骨骼的名称，以及 X / Z 的大小。head 将是骨骼从其中心向前方向的深度，tail 相同但面向后方（tail 通常需要是负值）。然后按'Add/Adjust bone'。

    您可以通过在此插件部分中输入其名称、编辑值并按'Add/Adjust Bone'按钮来更改任何辅助骨骼的大小。

    现在您必须将骨骼移动到其正确位置。对于 "root"，它必须是实体底部的确切中心（"root"的原点/枢轴将在地面级别，因此其一半将在网格下方）。 我们可以使用 Transform 菜单更准确地将它放置在正确的位置。

  5. 一旦所有原点都正确，转到'File > Export > Stonehearth (.json)'并确保选中左下角的"Skeleton"： 为骨骼文件命名并点击"Export File"。

  6. 在您保存的位置查找您的骨骼文件，并确保删除无效的 JSON 并添加缺少的字段以指向动画和效果文件夹，以便游戏可以正确加载它。Blender 插件可以在没有这些字段的情况下工作，但如果我们这样保留它，游戏将无法加载文件：

创建 meta 文件

meta 文件将使在 Blender 中为我们的自定义实体制作动画更加方便。它可以手动准备或使用插件导出它。以下是在 Blender 中创建它的步骤：

  1. 在 Blender 中导入您的自定义模型的 .obj 文件（如果尚未导入）。

  2. 检查右上角层次结构中的骨骼名称是否正确，以防它们未正确导出（如果尚未检查）。

  3. 导入骨骼并点击"Prepare skeleton"。

  4. 调整任何辅助骨骼的大小，以便它们在制作动画时不会妨碍视图（骨骼只保存了它们的位置/原点）。

  5. 现在我们需要定义骨骼的层次结构：

    要做到这一点，首先右键单击子骨骼，然后按住 Shift 并右键单击父骨骼，然后按 Ctrl+P 以便我们看到'Set Parent To'菜单： 然后点击'Object'选项。例如，我们首先选择了腿，然后是身体，并且以该顺序同时选择了两者，我们设置了层次结构。现在当我们尝试移动身体时，腿将跟随移动。确保最终所有骨骼都是 "root" 的后代（直接或间接）。

  6. 一旦所有层次结构都正确，转到'File > Export > Stonehearth (.json)'并确保选中左下角的"Meta"： 为 meta 文件命名并点击"Export File"。

  7. 尝试加载您的骨骼和 meta 文件并准备动画。现在尝试移动一个骨骼，看看子骨骼是否随它移动。

为自定义实体创建动画

工作流程与为现有 rigs 创建动画相同，只不过我们首先需要如上所述自己创建骨骼和 meta 文件。然后我们可以重用它们来制作任何动画。

如果我们创建了几个共享相同骨骼但不共享枢轴的 rigs，也可以应用偏移。
