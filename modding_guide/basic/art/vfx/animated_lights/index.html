灯光效果

Stonehearth 中的灯光通常是循环效果。我们可以将它们与 cubemitter 轨道一起使用，以使灯光更加明显。

效果文件

这些是效果文件中的示例轨道：

  "light": {
     "type": "light",
     "light": "/rayyas_children/data/horde/animatedlights/fire/hearth/hearth.lamp.animatedlight.json",
     "transforms": { "x": 3, "y": 3, "z": 0 },
     "shadows": true
  },
  "light2": {
     "type": "light",
     "light": "/rayyas_children/data/horde/animatedlights/fire/hearth/hearth.lamp.animatedlight.json",
     "transforms": { "x": -3, "y": 3, "z": 0 },
     "shadows": true
  }

它们的属性很少：

  • "type" : "light" -- 轨道的类型（灯光效果）。

  • "light" -- 定义灯光设置的 JSON 文件的路径。与 cubemitter 文件一样，默认情况下它们将从 stonehearth/data/horde 读取。

    在现有效果的灯光轨道中，您会看到路径以 animatedlights 文件夹开头。它实际上将被读取为 stonehearth/data/horde/animatedlights/lamp/lamp.animatedlight.json，因此我们的路径应从我们 mod 的命名空间开始，例如 /candyland/data/horde/animatedlights/cupcake_lamp/cupcake_lamp.animatedlight.json。否则游戏将尝试从 stonehearth/data/horde 读取它们。

  • "transforms": {} -- 包含 "x"、"y" 和 "z" 字段，用于定义放置灯光原点的偏移量（以方块数为单位）。

  • "shadows" -- 可选字段（默认为 false）。一个布尔值，用于启用/禁用从此光源投射阴影。

    阴影在计算上很昂贵，因此如果您的灯光很小，您可以在此处使用 false。对于更大的灯光，如火坑，看到它们从周围物体投射阴影是很好的（实际阴影数量仍然可以从游戏中的图形设置中调整）。

  • "bone" -- 可选字段。它是我们要附加此灯光的实体骨骼的名称。

创建动画灯光文件

animatedlight JSON 文件的示例：

  {
     "name": "fire",
     "duration": 1800,
     "loops": true,
     "intensity": {
        "over_lifetime": {
           "kind": "CONSTANT",
           "values": [0.8]
        }
     },
     "radius": {
        "start": {
           "kind": "CONSTANT",
           "values": [10]
        }
     },
     "color": {
        "start": {
           "kind": "CONSTANT",
           "values": [0.89, 0.227, 0.085]
        }
     }
  }

我们有常见的 "name"、"duration"（以毫秒为单位）和 "loops" 属性。动画灯光通常循环。

然后我们有这 3 个属性：

  • "intensity" -- 灯光的强度。值可以高于 1。
  • "radius" -- 灯光的半径。灯光将在边缘淡出，因此可能感觉不精确。
  • "color" -- 灯光的颜色。它与地形颜色混合。
  • "inner_radius" -- 可选属性。它将给予灯光中心更多强度，感觉像是两个灯光在同一位置，一个比另一个小，具体取决于此值。

所有这些属性都可以有 "start" 和/或 "over_lifetime" 字段。"start" 可以是 "CONSTANT" 或 "RANDOM_BETWEEN"，"over_lifetime" 可以是 "CONSTANT"、"RANDOM_BETWEEN"、"CURVE" 或 "RANDOM_BETWEEN_CURVES"。有关如何根据"kind"设置"values"的示例，请参见此表。

如何在 SHED 中创建灯光效果

在编写本指南时，如果效果不在上述目录内，SHED 将无法检测到它们。 因此，请确保您使用与 stonehearth mod 相同的目录结构（your__mod_namespace/data/horde/animatedlights）。

转到 Effects Editor，选择'Lights'选项卡并克隆现有灯光：

然后在右侧的 Web (CEF) 编辑器或中间的 JSON 编辑器中编辑值。有关曲线编辑器的一些提示可以在这里找到。
