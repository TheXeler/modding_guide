艺术工作流程

在本节中，我们将解释如何为您的 mod 创建艺术资源、导出它们的设置，以及如何在您的 mod 中引用它们。

目录

  1. 创建 3D 模型：如何为您的实体创建和导出 .qb 模型。
  2. 创建 2D 资源：如何生成要在 UI 中显示的图标和肖像。
  3. VFX（视觉效果）：如何创建和添加不同类型的效果。
    1. 添加动画：如何为您的实体添加动画效果。
      1. 对于现有实体：为 stonehearth mod 中的现有实体创建动画的工作流程。
      2. 对于自定义实体：为您的自定义实体创建动画的工作流程。
    2. 粒子效果：如何创建粒子效果。
    3. 光照效果：如何创建动画光照效果。
    4. 其他效果：如何创建其他类型的效果，例如音效等。
