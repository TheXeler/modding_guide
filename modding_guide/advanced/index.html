高级 Modding

在本节中，我们将解释高级 modding 主题，面向那些想要了解如何修改涉及更多编程的游戏部分的玩家。我们将涵盖组件、AI 等的 Lua 文件，以及 UI 文件。

目录

  1. 用户设置和日志记录：如何链接在 user_settings.json 中声明的选项以便在您的 mod 中使用它们，以及如何从您的 Lua 文件打印到 stonehearth.log。
  2. 事件：如何从 Lua 文件触发和监听事件。
  3. Lua 脚本：什么是保存变量和观察者。Lua 脚本的生命周期函数。
    1. 组件：如何创建您自己的组件和组件渲染器。
    2. 控制器：如何创建您自己的控制器。
  4. AI 动作：Stonehearth 中 AI 的工作原理以及如何向游戏添加您自己的 AI 动作。
  5. 服务：如何向游戏添加您自己的服务。
  6. 调用处理器：什么是调用处理器以及如何使用它们。
  7. UI：如何创建新的 UI 视图。
  8. C++ 函数：我们可以从 Javascript 和 Lua 调用的 C++ 组件和函数列表。
