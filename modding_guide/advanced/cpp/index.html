C++ 函数

您可以在此处找到我们可以从 Lua/Javascript 调用的现有 C++ 函数列表：PDF。

大多数函数不包含可能的参数，但尝试调用它们，可能的签名应该会出现在错误消息中。

  • 从 Javascript 调用 C++ 函数
  • 从 Lua 调用 C++ 函数

从 Javascript 调用 C++ 函数

我们可以使用 radiant:call 以及路由名称和我们要传入的参数从 Javascript 调用客户端函数：

  radiant.call('radiant:route_name', arg1, arg2, {arg3: {'x':1}{'y':2}{'z':3}} );

例如：

  radiant.call('radiant:play_sound', {'track' : 'stonehearth:sounds:ui:action_click' });

从 Lua 调用 C++ 函数

如果某个函数已对 Javascript 可用，也可以从 Lua 调用它。使用以下语法：

  _radiant.call('radiant:game:get_game_speed')
     :done(function(r)
         -- 做一些事情
      end)

我们也可以访问 C++ 组件并使用 PDF 中列出的函数：

  local mob = entity:get_component('mob')
  local facing = mob and mob:get_facing()

调用 PDF 中 Open Lua 函数列表中的函数将是这样的：

  _radiant.sim.topology.are_connected(entity, pt)
